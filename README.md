# 🌾 About JeevanMitra AI

JeevanMitra AI is an AI-powered farming assistant built to help farmers make better agricultural decisions using modern Artificial Intelligence.

The platform provides intelligent crop recommendations, disease detection, crop yield prediction, live market insights, multilingual voice assistance, and region-specific farming guidance—all directly in the browser without requiring a backend server.

Designed with accessibility and simplicity in mind, JeevanMitra AI works on desktops, tablets, and low-end smartphones, making advanced agricultural technology available to everyone.

# 🎯 Problem Statement

Millions of farmers make critical agricultural decisions with limited access to timely information.

JeevanMitra AI helps farmers answer questions like:

- Which crop should I grow?
- Is my crop diseased?
- What yield can I expect?
- What are today's market prices?
- When should I sow or harvest?
- Can I get advice in my own language?
