🌱 JeevanMitra AI

JeevanMitra AI is an intelligent agriculture assistant that helps farmers with crop recommendations, disease detection, and real-time farming insights using AI.

---

🚀 Features

- 🌾 Crop Recommendation based on conditions
- 🦠 Disease Detection using AI
- 🌦️ Weather-based insights
- 📊 Smart farming suggestions
- 🤖 AI chatbot support

---

🛠️ Tech Stack

- Frontend: HTML, CSS
- Backend: Python
- AI Models: Groq API (LLaMA, Gemma models)
- Tools: VS Code, GitHub

---

⚙️ How to Run Locally

1. Clone the repository

git clone https://github.com/simha05/jeevanmitra-ai.git

2. Navigate to project folder

cd jeevanmitra

3. Install dependencies

python -m pip install -r requirements.txt

4. Add your API key in "backend/config.py"

GROQ_API_KEY = "your_api_key_here"

5. Run the project

python run.py

6. Open in browser

http://localhost:8080
